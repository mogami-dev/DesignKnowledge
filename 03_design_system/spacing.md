# Spacing

## 基本方針

8pt Gridを基本とする。

## Tokens

| Token | Value | Usage |
|---|---:|---|
| xs | 4 | very small gap |
| sm | 8 | small gap |
| md | 16 | standard gap |
| lg | 24 | section gap |
| xl | 32 | large section gap |
| xxl | 48 | page-level gap |

## 確認観点

- 画面端の余白は十分か
- 関連する情報が近くにあるか
- 関連しない情報が離れているか
- ボタン周りに押しやすい余白があるか
