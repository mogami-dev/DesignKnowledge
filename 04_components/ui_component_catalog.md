# UI Component Catalog

## Button

用途:

- 画面遷移
- 保存
- 削除
- 状態変更

種類:

- Primary
- Secondary
- Danger
- Text
- Icon
- FAB

## Card

用途:

- 一覧表示
- 情報グループ
- 画像表示

注意:

- カードを多用しすぎない
- 写真一覧ではカードUIが一覧性を落とす場合がある

## Navigation Bar

要素:

- 戻る
- タイトル
- Menu
- Action

## Bottom Sheet

用途:

- 確認
- 選択
- 編集
- 次アクション提示

設計項目:

- 高さ
- Dismiss可否
- Primary Action
- Secondary Action

## Dialog

用途:

- 破壊的操作の確認
- 重要なエラー
- ユーザー判断が必要な状態

## Toast

用途:

- 保存完了
- コピー完了
- 通信成功

## Snackbar

用途:

- Undo可能な操作
- 一時通知

## Search Bar

用途:

- 一覧検索
- タグ検索
- 絞り込み

## Tag

用途:

- ラベル
- カテゴリ
- フィルタ

## Chip

用途:

- 単一選択
- 複数選択
- 条件指定

## List Item

要素:

- アイコン
- タイトル
- 説明
- 右矢印
- Switch

## Image

種類:

- Thumbnail
- Full
- Zoom
- Placeholder

## Empty State

要素:

- Icon
- Message
- CTA

## Loading

種類:

- Skeleton
- Spinner
- Progress

## Error View

要素:

- Message
- Retry
- Support

## Form

種類:

- Text
- Number
- Date
- Picker
- Toggle
- Stepper

## Camera

要素:

- Preview
- Guide
- Capture
- AI補正
- Retake
- Confirm
